<Steps>
<Step>
Install the Wheel Picker via the nocta-ui CLI:

```bash lineNumbers title="Terminal"

```
</Step>
<Step>
Import the components you need:

```tsx lineNumbers title="Import"

  WheelPicker,

  WheelPickerItem,
```

## API Reference

### WheelPicker

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | — | Controlled value |
| `defaultValue` | `string` | — | Default value when uncontrolled |
| `onValueChange` | `(value: string) => void` | — | Fired after the selection changes |
| `disabled` | `boolean` | `false` | Disables focus and gesture interactions |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Controls item height and component width |
| `visibleCount` | `number` | `20` | Number of virtual segments used for the 3D wheel layout |
| `infinite` | `boolean` | `false` | Enables looping scroll that wraps around |
| `className` | `string` | `''` | Additional CSS classes |
| `aria-label` | `string` | — | Accessible label when no visible label present |
| `children` | `React.ReactNode` | — | Picker options rendered as `WheelPickerItem` children |

### WheelPickerItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | — | Item value |
| `children` | `React.ReactNode` | — | Custom item content (label) |
| `className` | `string` | `''` | Additional CSS classes |

### WheelPickerGroup

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Shared size passed to nested pickers |
| `disabled` | `boolean` | `false` | Disables interactions for all nested pickers |
| `contentClassName` | `string` | `''` | Classes for the inner flex container that holds each picker |
| `className` | `string` | `''` | Additional CSS classes applied to the group frame |
| `children` | `React.ReactNode` | — | Typically multiple `WheelPicker` instances |

All components also accept their respective HTML element attributes.
