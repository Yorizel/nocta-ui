## API Reference

### Toggle

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `pressed` | `boolean` | `undefined` | Controlled pressed (checked) state |
| `defaultPressed` | `boolean` | `false` | Initial state for uncontrolled usage |
| `onPressedChange` | `(pressed: boolean) => void` | - | Callback fired when the pressed state changes |
| `variant` | `'default'` | `'default'` | Visual variant of the toggle |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Size of the toggle |
| `disabled` | `boolean` | `false` | Disables the toggle when true |
| `className` | `string` | `''` | Additional classes for custom styling |
| `children` | `React.ReactNode` | `undefined` | Toggle content (text, icon, etc.) |

The component forwards all other props from `@ariakit/react`'s Checkbox. Internally, it renders as a `<button>` element via the `render={<button />}` prop for visual flexibility.
