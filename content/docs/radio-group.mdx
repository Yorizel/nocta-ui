## API Reference

### RadioGroup

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | — | Controlled value of the selected option |
| `defaultValue` | `string` | — | Initial selected value for uncontrolled usage |
| `onValueChange` | `(value: string | number | null) => void)` | — | Callback fired when the selection changes |
| `disabled` | `boolean` | `false` | Disables selection for all radio items |
| `className` | `string` | `''` | Additional CSS classes applied to the group wrapper |
| `children` | `React.ReactNode` | — | One or more `RadioGroupItem` components |

### RadioGroupItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | — | Value associated with the radio option |
| `label` | `React.ReactNode` | — | Main label displayed next to the indicator |
| `description` | `React.ReactNode` | — | Optional helper text below the label |
| `disabled` | `boolean` | `false` | Disables this individual option |
| `className` | `string` | `''` | Additional CSS classes applied to the label wrapper |

All other props are forwarded to the underlying Ariakit component.
