## Multiple Selection

Remove the `single` prop to allow toggling several items at the same time. The component manages an array of selected values.

## Sizes

Toggle groups support `sm`, `md`, and `lg` sizes. The group and its items inherit the size for consistent spacing.

<DocsTabs defaultValue="preview">
  <DocsTab value="preview" lineNumbers title="Preview">
    <div className="my-6">
      <ToggleGroupSizesDemo />
    </div>
  </DocsTab>
  <DocsTab value="code" lineNumbers title="Code">
    ```tsx lineNumbers title="Toggle Group Sizes"
    const [values, setValues] = useState({ sm: ['left'], md: ['center'], lg: ['right'] });

    <ToggleGroup single size="sm" value={values.sm} onValueChange={(next) => setValues(prev => ({ ...prev, sm: next }))}>
      <ToggleGroupItem value="left">Left</ToggleGroupItem>
      <ToggleGroupItem value="center">Center</ToggleGroupItem>
      <ToggleGroupItem value="right">Right</ToggleGroupItem>
    </ToggleGroup>
    <ToggleGroup single size="md" value={values.md} onValueChange={(next) => setValues(prev => ({ ...prev, md: next }))}>
      <ToggleGroupItem value="left">Left</ToggleGroupItem>
      <ToggleGroupItem value="center">Center</ToggleGroupItem>
      <ToggleGroupItem value="right">Right</ToggleGroupItem>
    </ToggleGroup>
    <ToggleGroup single size="lg" value={values.lg} onValueChange={(next) => setValues(prev => ({ ...prev, lg: next }))}>
      <ToggleGroupItem value="left">Left</ToggleGroupItem>
      <ToggleGroupItem value="center">Center</ToggleGroupItem>
      <ToggleGroupItem value="right">Right</ToggleGroupItem>
    </ToggleGroup>
    ```
  </DocsTab>
</DocsTabs>

## API Reference

### ToggleGroup

| Prop            | Type                        | Default     | Description                                                 |
| --------------- | --------------------------- | ----------- | ----------------------------------------------------------- |
| `single`        | `boolean`                   | `false`     | Determines whether only one item can be selected at a time. |
| `value`         | `string[]`                  | `undefined` | Controlled selected values.                                 |
| `defaultValue`  | `string[]`                  | `[]`        | Initial selection for uncontrolled usage.                   |
| `onValueChange` | `(value: string[]) => void` | -           | Called when the selection changes.                          |
| `variant`       | `'default'`                 | `'default'` | Outer container style.                                      |
| `size`          | `'sm' \| 'md' \| 'lg'`      | `'md'`      | Defines the toggle size and padding.                        |
| `disabled`      | `boolean`                   | `false`     | Disables all toggle items when true.                        |
| `className`     | `string`                    | `''`        | Additional classes on the group container.                  |

### ToggleGroupItem

| Prop        | Type              | Default             | Description                                      |
| ----------- | ----------------- | ------------------- | ------------------------------------------------ |
| `value`     | `string`          | -                   | Unique identifier for the item within the group. |
| `disabled`  | `boolean`         | Inherits from group | Disables only this item when true.               |
| `className` | `string`          | `''`                | Additional classes for custom styling.           |
| `children`  | `React.ReactNode` | -                   | Content such as text or icons.                   |
